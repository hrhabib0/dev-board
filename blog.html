Question-1: What are the different ways to select an element in the DOM?
We can select an element in different ways in DOM, such as getElementsByTagName(), getElementsByClassName(), getElementById(), querySelectorAll() and querySelector().
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML is include html tag and content, innertText is include only text of an element and textContent show element's text and style.
Question-3: What is event delegation in the DOM?
Event delegation is a technique to handle event in parent with event bubbling and event capturing.
Question-4: What is event bubbling in the DOM?
Event bubbling in the DOM is a method where event trigerred first on child element then it bubbling consequently to the parent.
Question-5: How do you create, add, and remove elements using JavaScript?
To create an element we can use this step : document.createElement(); and to remove an element we can use removeChild() method.
